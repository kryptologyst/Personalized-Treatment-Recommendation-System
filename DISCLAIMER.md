## SAFETY FEATURES

### De-identification

The system includes automatic de-identification features to protect patient privacy:

- **Phone Numbers**: Automatically detected and replaced with `[PHONE]`
- **Social Security Numbers**: Detected and replaced with `[SSN]`
- **Dates**: Medical dates replaced with `[DATE]`
- **Email Addresses**: Replaced with `[EMAIL]`
- **Custom Patterns**: Configurable regex patterns for additional PHI

### Input Validation

- **Text Length Limits**: Prevents excessively long inputs
